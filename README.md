Este projeto é uma API REST de e-commerce desenvolvida com Flask, utilizando PostgreSQL como banco de dados e SQLAlchemy como ORM.

Atualmente a API possui funcionalidades básicas como:

- Adicionar produtos (POST)
- Listar produtos (GET)
- Deletar produtos (DELETE)

O projeto utiliza Flask-Migrate para controle de versão do banco de dados.

Os testes dos endpoints foram realizados com Insomnia.

Este projeto está em desenvolvimento e será atualizado com novas funcionalidades nos próximos dias.
